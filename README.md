# ПОЯСНИТЕЛЬНАЯ ЗАПИСКА
## к расчётно-графической работе по дисциплине «Программирование»

**Тема:** Создание программного средства «Encryption Algorithm RGR»

## ВВЕДЕНИЕ

Пояснительная записка описывает разработку программного средства «Encryption Algorithm RGR», реализующего шифрование и дешифрование текстовых данных и файлов с использованием динамически подключаемых библиотек криптографических алгоритмов.

Работа выполнена в рамках расчётно-графической работы (РГР) по дисциплине «Программирование» на кафедре «Защиты информации» ФГБОУ ВО «НГТУ».

Цель работы — практическое изучение принципов модульного программирования, работы с динамическими библиотеками и основ криптографической защиты информации.

## 1. НАЗНАЧЕНИЕ И ОБЛАСТЬ ПРИМЕНЕНИЯ

### 1.1 Назначение программы

Программное средство «Encryption Algorithm RGR» предназначено для:
- шифрования и дешифрования текстовых данных;
- шифрования и дешифрования файлов (текстовых и бинарных);
- генерации криптографических ключей;
- демонстрации работы алгоритмов шифрования.

### 1.2 Область применения

Программа используется в учебных и демонстрационных целях для изучения:
- принципов работы динамически подключаемых библиотек;
- основ криптографической защиты информации;
- разработки модульных программных систем;
- обработки файловых потоков и исключений.
